# 待办清单 + 事项提醒应用

一个功能完整、界面精美的待办事项管理应用，支持提醒功能、优先级设置和数据导入导出。

## ✨ 功能特性

### 核心功能
- ✅ **添加待办事项** - 快速添加任务，设置提醒时间和优先级
- ✅ **标记完成** - 一键标记任务完成状态
- ✅ **编辑事项** - 随时修改待办事项内容
- ✅ **删除事项** - 删除不需要的任务
- ✅ **清除已完成** - 批量清理所有已完成事项

### 高级功能
- 🔔 **事项提醒** - 设置提醒时间，浏览器通知提醒
- 🎯 **优先级设置** - 三级优先级（高/中/低），视觉区分
- 🔍 **智能筛选** - 全部/待完成/已完成 快速筛选
- 📊 **统计信息** - 实时显示任务统计数据
- 💾 **数据持久化** - LocalStorage 本地存储，刷新不丢失
- 📤 **数据导出** - 导出 JSON 格式备份
- 📥 **数据导入** - 导入备份文件恢复数据

### UI/UX
- 🎨 **精美设计** - 渐变色背景，圆角卡片
- 📱 **响应式布局** - 完美适配手机、平板、电脑
- ⚡ **流畅动画** - 悬停、点击动画效果
- 🌈 **优先级颜色** - 不同颜色区分优先级
- 🔔 **通知提醒** - 浏览器原生通知支持

## 🚀 使用方法

### 1. 直接打开
```bash
# 用浏览器打开 index.html 文件
open index.html  # macOS
start index.html  # Windows
xdg-open index.html  # Linux
```

### 2. 本地服务器（推荐）
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js http-server
npx http-server -p 8000

# 使用 PHP
php -S localhost:8000
```

然后访问 http://localhost:8000

## 📱 使用说明

### 添加待办事项
1. 在输入框中输入任务内容
2. （可选）设置提醒时间
3. （可选）选择优先级（低/中/高）
4. 点击"添加"按钮或按 Enter 键

### 管理待办事项
- **标记完成**：点击任务左侧的复选框
- **编辑任务**：点击 ✏️ 编辑按钮
- **删除任务**：点击 🗑️ 删除按钮
- **筛选查看**：使用顶部的筛选按钮

### 设置提醒
1. 添加或编辑任务时，点击"选择日期时间"
2. 选择提醒的日期和时间
3. 保存后，系统会在指定时间发送浏览器通知

> 💡 **提示**：首次使用时，浏览器会请求通知权限，请点击"允许"

### 数据管理
- **导出数据**：点击"导出数据"按钮，下载 JSON 备份文件
- **导入数据**：点击"导入数据"按钮，选择之前导出的 JSON 文件

## 🎨 自定义

### 修改配色
编辑 `style.css` 中的渐变色：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 修改优先级颜色
```css
.todo-item.priority-high {
    border-left-color: #ff4757; /* 红色 */
}
.todo-item.priority-medium {
    border-left-color: #ffa502; /* 黄色 */
}
.todo-item.priority-low {
    border-left-color: #2ed573; /* 绿色 */
}
```
