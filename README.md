# 待办清单 + 事项提醒 v2.1

一个功能完整、界面精美的待办事项管理应用，支持标签页切换、番茄时钟和定期任务。

## ✨ 功能特性

### 🎯 标签页界面（v2.1 新增）
- 📋 **待办事项标签** - 管理日常任务
- 🍅 **番茄时钟标签** - 专注工作模式
- 🔄 **定期任务标签** - 自动重复任务
- ✨ **平滑切换** - 优雅的过渡动画

### 📋 待办事项管理
- ✅ 添加/编辑/删除待办事项
- ✅ 优先级设置（高/中/低）
- ✅ 智能筛选（全部/待完成/已完成）
- ✅ 提醒功能（浏览器通知）
- ✅ 重复周期设置

### 🍅 番茄时钟
- ⏱️ 25分钟工作模式
- ☕ 5分钟休息模式
- ▶️ 开始/暂停/重置控制
- 🔄 模式切换功能
- 🔔 音频提示音
- 📊 番茄工作法指南

### 🔄 定期任务
- 📅 每天重复任务
- 📆 每周重复任务
- 🗓️ 每月重复任务
- 📅 每年重复任务
- 🔄 自动创建下一次
- 📋 独立管理界面

### 💾 数据管理
- LocalStorage 本地存储
- JSON 格式导出导入
- 自动保存功能

## 🚀 使用方法

### 访问应用
**在线地址**：https://joshualover.github.io/todo-reminder-app/

### 标签页使用

1. **待办事项标签**
   - 点击"待办事项"标签
   - 添加、编辑、完成任务
   - 设置提醒和重复周期

2. **番茄时钟标签**
   - 点击"番茄时钟"标签
   - 点击"▶️ 开始"启动25分钟倒计时
   - 专注工作，完成后自动切换到休息模式
   - 参考"番茄工作法"指南使用

3. **定期任务标签**
   - 点击"定期任务"标签
   - 输入任务内容
   - 选择重复周期（每天/每周/每月/每年）
   - 点击"添加"按钮
   - 系统会自动按时创建待办事项

## 🎯 番茄工作法使用技巧

1. **选择任务** - 从待办列表中选择一个任务
2. **启动番茄钟** - 点击"开始"按钮
3. **专注工作** - 25分钟内专注该任务
4. **短暂休息** - 时间到后休息5分钟
5. **重复循环** - 完成4个番茄钟后，长休息15-30分钟

## 🔄 定期任务示例

- **每天**："晨跑30分钟"、"背诵10个单词"
- **每周**："写周报"、"整理书桌"
