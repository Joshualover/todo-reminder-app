# 待办清单 + 事项提醒 v2.0

一个功能完整、界面精美的待办事项管理应用，支持提醒功能、优先级设置、定期任务和番茄时钟。

## ✨ 功能特性

### 核心功能
- ✅ **添加待办事项** - 快速添加任务，设置提醒时间和优先级
- ✅ **标记完成** - 一键标记任务完成状态
- ✅ **编辑事项** - 随时修改待办事项内容
- ✅ **删除事项** - 删除不需要的任务
- ✅ **智能筛选** - 全部/待完成/已完成 快速筛选

### 🍅 番茄时钟（新功能 v2.0）
- ⏱️ **25分钟工作模式** - 专注工作25分钟
- ☕ **5分钟休息模式** - 短暂休息放松
- ▶️ **开始/暂停/重置** - 完整的计时控制
- 🔄 **模式切换** - 工作和休息模式自由切换
- 🔔 **音频提示** - 完成时播放提示音
- 📊 **标题倒计时** - 浏览器标签显示剩余时间

### 🔄 定期任务（新功能 v2.0）
- 📅 **每天重复** - 每天自动创建任务
- 📆 **每周重复** - 每周自动创建任务
- 🗓️ **每月重复** - 每月自动创建任务
- 📅 **每年重复** - 每年自动创建任务
- 🔄 **自动生成** - 完成当前任务后自动创建下一次
- 📋 **独立管理** - 专门的定期任务列表

### 高级功能
- 🔔 **事项提醒** - 设置提醒时间，浏览器通知提醒
- 🎯 **优先级设置** - 三级优先级（高/中/低），视觉区分
- 💾 **数据持久化** - LocalStorage 本地存储，刷新不丢失
- 📤 **数据导出** - 导出 JSON 格式备份
- 📥 **数据导入** - 导入备份文件恢复数据

### UI/UX
- 🎨 **精美设计** - 渐变色背景，圆角卡片
- 📱 **响应式布局** - 完美适配手机、平板、电脑
- ⚡ **流畅动画** - 悬停、点击动画效果
- 🌈 **优先级颜色** - 不同颜色区分优先级
- 🔔 **通知提醒** - 浏览器原生通知支持

## 🚀 使用方法

### 1. 访问应用
**在线地址**：https://joshualover.github.io/todo-reminder-app/

### 2. 添加待办事项
1. 在输入框中输入任务内容
2. （可选）设置提醒时间
3. （可选）选择优先级（低/中/高）
4. （可选）选择重复周期
5. 点击"添加"按钮或按 Enter 键

### 3. 使用番茄时钟
1. 点击"▶️ 开始"启动25分钟倒计时
2. 专注工作，倒计时结束会听到提示音
3. 自动切换到5分钟休息模式
4. 休息结束后继续下一个番茄钟
5. 随时可以暂停、重置或切换模式

### 4. 设置定期任务
1. 在"定期任务"区域输入任务内容
2. 选择重复周期（每天/每周/每月/每年）
3. 点击"添加"按钮
4. 系统会自动按时创建待办事项

### 5. 管理待办事项
- **标记完成**：点击任务左侧的复选框
- **编辑任务**：点击 ✏️ 编辑按钮
- **删除任务**：点击 🗑️ 删除按钮
- **筛选查看**：使用顶部的筛选按钮

### 6. 数据管理
- **导出数据**：点击"导出数据"按钮，下载 JSON 备份文件
- **导入数据**：点击"导入数据"按钮，选择之前导出的 JSON 文件

## 💡 使用技巧

### 番茄工作法
1. 选择一个待完成的任务
2. 启动番茄时钟（25分钟）
3. 专注工作直到时钟结束
4. 休息5分钟
5. 每4个番茄钟后，休息15-30分钟

### 定期任务示例
- **每天**："晨跑30分钟"、"背诵10个单词"
- **每周**："写周报"、"整理书桌"
- **每月**："支付账单"、"备份电脑"
- **每年**："年度体检"、"更新证件"
