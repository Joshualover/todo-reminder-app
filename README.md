# 待办清单 v2.1

一个简洁高效的待办事项管理应用，采用移动端优先的底部导航设计。

## ✨ 功能特性

### 📱 底部导航设计
- **📋 待办** - 管理日常任务
- **🍅 番茄** - 专注工作模式
- **🔄 定期** - 自动重复任务

### 📋 待办事项
- 添加/编辑/删除待办事项
- 优先级设置（高/中/低）
- 智能筛选（全部/待完成/已完成）
- 提醒功能
- 重复周期设置

### 🍅 番茄时钟
- 25分钟工作模式
- 5分钟休息模式
- 开始/暂停/重置控制
- 音频提示

### 🔄 定期任务
- 每天重复任务
- 每周重复任务
- 每月重复任务
- 每年重复任务
- 自动创建下一次

## 🚀 使用方法

访问应用：https://joshualover.github.io/todo-reminder-app/

## 📊 技术栈

- HTML5 + CSS3 + JavaScript ES6+
- LocalStorage 本地存储
- 移动端优先设计
